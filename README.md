# areyouOK
大家好，我是练习时长两年半的AI对话编程力工

## 简介

这是一个用 Rust 编写的跨平台窗口应用程序，窗口标题为 "areyouOK"。

## 依赖

- Rust (推荐使用最新稳定版)
- Cargo

## 构建

```bash
cargo build
```

## 运行

```bash
cargo run
```

## 平台兼容性

- ✅ Windows
- ✅ Linux (需要 X11 或 Wayland 支持)
- ✅ macOS

## 技术栈

- [winit](https://github.com/rust-windowing/winit) - 跨平台窗口创建和事件处理库
